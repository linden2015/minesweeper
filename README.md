# Minesweeper

A minesweeper game in Java.

![image](https://s29.postimg.org/6h87eechz/minesweeper_0_1.png)

## Instructions

Run as follows:

    mvn exec: java

## Requirements (not tested with earlier)

* Maven 3.3.9
* Java 1.8
